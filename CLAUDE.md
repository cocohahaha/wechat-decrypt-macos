## Project: wechat-decrypt-macos

微信 macOS 聊天记录解密与分析 MCP Server。

### Quick Start
- Install: `bash install.sh`
- Key extraction requires WeChat to be running
- MCP Server: `server.py` (~440 lines, FastMCP)

### Architecture
- Single file: `server.py`
- Encryption: SQLCipher 4, cipher_page_size=4096
- Key extraction: Mach VM API (task_for_pid -> memory scan -> codec_ctx)
- Protocol: MCP (Model Context Protocol) via FastMCP

### Tools provided
- `wechat_list_chats`: List all conversations
- `wechat_read_chat`: Read messages with a contact
- `wechat_search_messages`: Search by keyword
- `wechat_recent_messages`: Recent messages overview
- `wechat_chat_summary`: Structured summary for analysis

### Key files
- `server.py` — MCP Server (FastMCP, all tools)
- `key.txt` — SQLCipher key (gitignored, generated by key extraction)
- `pyproject.toml` — Package config, depends on `mcp[cli]`
- `install.sh` — One-click setup script
